# Security Policy

## Supported Versions

We take the security of promptkit seriously, especially since it is intended to be used as a guardrail layer for LLM applications.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability (for example in guardrail bypasses, PII leaks, or injection vectors), please report it responsibly.

**Do not open a public issue.**

Email: contact@example.com (or open a private security advisory on GitHub if supported).

Please include:
- Description of the issue
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan as quickly as possible.

Thank you for helping keep the LLM tooling ecosystem safer.
